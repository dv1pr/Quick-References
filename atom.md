## Atom Quick Reference

### Navigation
**Toggle between tree view**
```
^0
```

**Move tabs left or right**
```
cmd + opt + <- (or ->)
```


### Creation
**New file (when in tree view)**
```
a
```

**New directory (when in tree view)**
```
shift + a
```


### Text Editing
**Select entire word**
```
cmd + d
```
