## Atom Keyboard Shortcuts

### Navigation
Toggle between tree view
```
^0
```

Move active  tab left or right
```
⌘ + opt + left/right
```


### Creation
New file (when in tree view)
```
a
```

New directory (when in tree view)
```
shift + a
```


### Text Editing/Manipulation
Select entire word
```
⌘ + d
```

Select current line
```
⌘ + l
```

Delete to beginning of word
```
option + backspace
```

Set bookmark
```
⌘ + F2
```

Go to next bookmark
```
F2
```
